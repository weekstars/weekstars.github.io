COMSOL波动光学模块 3

波动光学建模

周期边界条件

波动光学模块具有专用的周期条件。周期条件可以识别出相等形状的源平面边界和目标边界上的简单映射。目标也可以相对于源进行旋转。有三种类型的周期条件（只有前两种用于瞬态分析）：

  • 连续性——解的切向分量在源和目标上相等。
  • 反周期性——切向分量具有相反的符号。
  • Floquet周期性——在切向分量之间存在相移。相移由波矢量和源与目标之间的距离决定。Floquet周期性通常用于涉及平面波与周期结构相互作用的模型。

周期边界条件必须具有兼容的网格。这可以通过在网格设置中启用物理控制的网格序列类型来自动实现，或通过手动设置正确的网格序列。

散射场

对于已知背景场在自由空间中照亮物体的问题，可以使用散射场公式。由于背景场的方程已知，它可以作为模型输入输入，不需要计算。从频域控制方程开始： \[ \nabla \times\left(\mu^{-1} \nabla \times \mathbf{E}\right)-\omega^{2} \varepsilon_{\mathrm{c}} \mathbf{E}=\mathbf{0} \] 总电场\(\mathbf{E}\)可以被分解为 \[ \mathbf{E}=\mathbf{E}_{\text {total }}=\mathbf{E}_{\text {background }}+\mathbf{E}_{\text {relative }} \] 已知的背景场成为一个源项，散射场公式因此求解了相对电场。可以指定一个线性偏振平面波背景场、一个傍轴近似高斯光束，或一个用户定义的背景场。当求解散射场公式时，总场、背景和相对电场是可得到的。相对场是背景场与总场之间的差值。它是参与远场计算的相对场。该方法的好处是，如果背景场的大小比散射场大得多，如果求解相对场，模拟的精度会得到提高。另一个优点是很容易在均匀介质建模域周围建立一个完美匹配的层。

这种方法的缺点是，相对场需要进行一些仔细的解释。相对电场在概念上可以分解为： \[ \mathbf{E}_{\text {relative }}=\mathbf{E}_{\text {scattered }}+\mathbf{E}_{\text {correction }}+\mathbf{E}_{\text {cancellation }} \] \(\mathbf{E}_{\text {scattered }}\)项是从物体散射的场，这是在散射问题中感兴趣的场。然而，相对场也可以包括代表对背景场的校正和对背景场的抵消的分量。\(\mathbf{E}_{\text {correction }}\)项在背景场不准确满足麦克斯韦方程组时非零，比如当背景场为傍轴近似高斯光束。\(\mathbf{E}_{\text {cancellation }}\)在总电场必须为零时非零且等于\(-\mathbf{E}_{\text {background }}\),比如在完全屏蔽的物体内部，或在一个相对较大的屏蔽物体后面。请注意，这种分解只是概念性的，可得到的只有相对场。

使用散射场公式的另一种方法是在使用端口时设置在内部端口上激活狭缝条件。然后，域可以被内部端口激发，输出场可以被完全匹配的层吸收。

远场计算的建模

例如，由天线辐射产生的远场可以由近场解在边界上使用远场分析得出。天线位于原点附近，而远场在无穷远处，但具有明确的角度方向\((\theta,\varphi)\)。通过计算以原点为中心的球体上远场的模，给出远场辐射模图案。球体表面上的每个坐标都代表一个角度方向。

电磁波，频域接口中的远场计算

电磁波，频域接口支持远场分析。要定义远场变量，请使用远场计算节点。选择一个域做远场计算。然后选择让算法积分近场的边界，输入远场变量名称，并指定在计算远场变量时模型中是否使用对称平面。对称平面必须与其中一个笛卡尔坐标平面重合。对于这些平面中的每一个，都可以选择要使用的对称类型，它可以是E（PMC）中的对称，也可以是H（PEC）中的对称。在这里进行选择，以匹配用于对称边界的边界条件。使用这些设置，由于对称性原因而不在模型中的几何部分可以包括在远场分析中。

如果在添加远场域特征之前定义了完全匹配层（PML）特征，则远场域和远场计算节点将自动获得选择。

对于输入的每个远场变量名称，COMSOL生成函数和变量，它们代表远场矢量的分量。这些变量的名称时在远场变量名称后添加自变量名称，例如输入名称Efar，几何是具有自变量\(x\)、\(y\)和\(z\)的笛卡尔坐标变量，生成的变量分别为Efarx、Efary和Efarz。另一方面，如果几何是具有自变量\(r\)、\(\varphi\)和\(z\)的轴对称的，则生成的变量被命名为Efarr、Efarphi和Efarz。

在二维空间中，该软件只为非零场分量生成变量。物理接口名称也出现在变量名的前面，通常看起来像ewfd.Efarz等等。对于每个生成的变量，相应都有一个具有相同名称的函数。这个函数将代表远场方向的矢量分量作为参数。

矢量分量也可以被解释为一个位置。例如，假设变量dx、dy和dz表示远场的方向。表达式Efarx(dx，dy，dz)给出了这个方向上的远场的值。要用角度表示方向，可以使用表达式Efarx(sin(theta)*cos(phi),sin(theta)*sin(phi),cos(theta))，其中变量theta和phi代表表示角度方向\((\theta,\varphi)\)。远场的大小和它用dB表示的值也分别作为变量normEfar和normdBEfar生成。

辐射图案

辐射图案可用来绘制全局变量的值（例如，远场模normEfar和normdBEfar，或远场变量Efar的分量）。为单位圆（2D）或单位球体（3D）上选定的角度绘制变量，可以手动指定角度间隔和角度数。对于二维辐射图案，也可以指定测量角度的参考方向和绘制平面的法向。对于三维辐射图案，还可以为曲面颜色指定一个表达式。

与制作折线图相比，辐射图案的主要优点是，用于定义绘图方向的单位圆/球体不是解的几何的一部分。因此，绘图方向的数量与解域的离散化解耦。
